# 🤖 AI Chatbot — FAQ Assistant
SoftGrowTech Internship · Task 2

An AI-powered chatbot built with Streamlit and Groq API (LLaMA 3.3-70B) that answers questions from any field.

## Features
- Answers questions from any field — Science, Math, History, Coding, Health, Law & more
- Multi-turn conversation with full memory
- Clean chat bubble UI
- Powered by Groq API — free and ultra-fast

## Tech Stack
- Framework: Streamlit (Python)
- AI Model: LLaMA 3.3-70B via Groq API
- Libraries: streamlit, requests

## Setup
pip install streamlit requests
Add your Groq API key in chatbot.py → API_KEY = "your_key_here"
streamlit run chatbot.py

Get a free API key at: https://console.groq.com
